# User Retention & Funnel Analysis

## Overview
This project analyses user onboarding behaviour and retention to identify drop-off points and improvement opportunities.

## Key Questions
- Where do users drop off in the onboarding funnel?
- How does retention change over time?
- What impacts user engagement?

## Tools
- Amplitude
- Funnel analysis
- Retention analysis

## Key Insights
- Identified key drop-off points in onboarding
- Low retention after initial interaction
- Opportunities to improve user activation

## Business Recommendations
- Optimise onboarding steps to reduce friction
- Improve early user engagement
- Monitor retention metrics regularly
